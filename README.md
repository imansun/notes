# یادداشت

مخزن یادداشت‌ها، ایده‌ها و پروژه‌های شخصی.

## هدف

اینجا جایی برای ثبت و سازماندهی یادداشت‌ها، نکات فنی، ایده‌های جدید و هر چیزی که در طول کار روی پروژه‌ها یاد می‌گیرم.

## ساختار

- `/docs` — مستندات و یادداشت‌های فنی
- `/scripts` — اسکریپت‌های کاربردی
- `/projects` — یادداشت‌های مربوط به پروژه‌های خاص
